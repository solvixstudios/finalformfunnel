Connect your WooCommerce store to Final Form — the premium order form builder for e-commerce.

== Description ==

Final Form for WooCommerce connects your store to the Final Form dashboard, allowing you to:

* Replace the default WooCommerce add-to-cart form with a premium, highly customizable order form
* Manage form assignments per product or store-wide from the Final Form dashboard
* Sync your WooCommerce product catalog with Final Form
* Track orders and analytics in one place

= How It Works =

1. Install and activate this plugin
2. Go to WooCommerce → Final Form in your WordPress admin
3. Generate an installation key from your [Final Form dashboard](https://app.finalform.app)
4. Paste the key into the plugin settings
5. Enable the form loader — done!

= Features =

* **One-click connection** — paste your key and you're connected
* **Automatic product sync** — your WooCommerce catalog is synced to Final Form
* **Smart script injection** — the loader only runs on product pages for optimal performance
* **Secure API** — all communication is authenticated via your installation key

== Installation ==

1. Upload the `finalform-woocommerce` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce → Final Form to configure
